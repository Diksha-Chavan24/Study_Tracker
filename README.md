# Study Tracker Application

This is a simple Java console-based application to track daily study activities.

## Features

* Add study logs
* View all logs
* Summary of study by date
* Summary of study by subject
* Export study data to CSV file

## Technologies Used

* Java
* Collection Framework (ArrayList, TreeMap)
* File Handling

## How to Run

1. Compile the program:
   javac program558.java

2. Run the program:
   java program558
